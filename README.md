# TinyScholar

TinyScholar is an AI-powered learning platform designed to make learning engaging and personalized for children.

## Vision

Instead of simply answering questions, TinyScholar acts as an intelligent learning companion by:

- 📚 Answering questions from uploaded study material
- 🎮 Gamifying learning through quizzes and rewards
- 📈 Tracking learning progress
- 🧠 Adapting to each child's pace

## Tech Stack

### Backend
- FastAPI
- Python
- Pydantic

### AI
- LlamaIndex
- ChromaDB
- Ollama / OpenAI

### Frontend
- React (planned)
